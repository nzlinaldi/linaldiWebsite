# Linaldi Multipage Website

This repository contains a multipage website for a stud welding supplier, focused on presenting the company, its services, and contact information with a modern and responsive design.

## Overview

This is a fully responsive multipage website built using HTML5, CSS3, and JavaScript.  
It was designed to provide a clean user interface for the company website while effectively presenting its services and products.

## Features

- Multipage structure  
- Responsive layout  
- Well-commented code  
- SEO-friendly structure  
- Performance optimization  
- Mobile and tablet navigation menu  
- Scroll reveal animations  
- Modern UI design  

## Technologies

- HTML5
- CSS3
- JavaScript
